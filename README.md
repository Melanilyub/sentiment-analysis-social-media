# Sentiment Analysis of Social Media Posts

## Project Overview
This project performs sentiment analysis on social media data using machine learning techniques.  
It classifies text into **positive**, **negative**, and **neutral** sentiments.

## Datasets
- Twitter US Airline Sentiment Dataset
- IMDB Movie Reviews Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib / Seaborn

## Process
1. Data Loading
2. Data Cleaning
3. Text Preprocessing
4. Feature Extraction (TF-IDF)
5. Model Training
6. Model Evaluation

## Models Used
- Logistic Regression
- Naive Bayes

## Results
- High accuracy on IMDB dataset (~89%)
- Moderate performance on Twitter dataset (~78%)

## Visualizations
- Sentiment distribution
- Confusion matrices
- Word cloud
